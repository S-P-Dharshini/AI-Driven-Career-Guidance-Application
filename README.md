# Grow like a Pro! Career Guidance Application

## How to Launch the Application

There are several ways to launch the application:

### Method 1: Using the Launcher HTML
1. Open the file "launcher.html" in your web browser
2. Click on "Launch Application" or any of the direct page links
3. The launcher will automatically redirect you to the application after 2 seconds

### Method 2: Using the Batch File
1. Double-click on "LaunchApp.bat"
2. The application will open in your default web browser

### Method 3: Creating a Desktop Shortcut
1. Double-click on "CreateShortcut.vbs"
2. A shortcut will be created on your desktop
3. Use this shortcut to launch the application anytime

### Method 4: Direct URL
You can also type or bookmark the following URL in your browser:
file:///C:/Career%20guidance%20application/index.html
